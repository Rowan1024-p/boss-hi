## 🚀 核心功能 (Core Features)

### 1. 🧠 三岗合一超级大脑 (Super Prompt)
摒弃了传统的“选岗位”模式。插件内置了**运营、供应链、开发**三大核心岗位的 JD 标准。
* AI 会自动阅读简历，判断候选人适合哪个岗位。
* 如果都不匹配，自动淘汰。

### 2. 👁️ 深度阅读技术 (Deep Reading)
拒绝“只看列表页摘要”的低效筛选。
* **模拟人类操作**：自动点击候选人卡片，等待侧边栏加载。
* **全量阅读**：支持读取 **15,000 字** 以内的完整简历文本（含项目经历、自我评价）。
* **精准判分**：有效避免因简历关键词折叠导致的误判。

### 3. 🛡️ 双重漏斗筛选 (Dual-Funnel Filter)
* **第一层（本地快筛）**：0成本过滤掉“活跃度低”、“学历不符”、“包含黑名单关键词（外包/兼职）”的简历。
* **第二层（AI 精筛）**：只有通过第一层的简历才会发送给 DeepSeek 进行深度评估，极大地节省 Token 成本。

### 4. 🔒 企业级隐私安全
* **0 数据上传**：所有配置、日志、筛选记录均存储在使用者浏览器的本地 (`chrome.storage.local`)。
* **去毒处理**：移除了原版中向第三方服务器发送手机号、配置信息的恶意代码。

## 🛠️ 安装与配置 (Installation)

### 第一步：配置 API Key
1.  打开项目文件夹下的 `background.js` 文件。
2.  找到第 5 行：
    ```javascript
    const DEEPSEEK_API_KEY = 'sk-xxxxxxxxxxxxxxxx'; 
    ```
3.  将 `'sk-...'` 替换为你购买的 **DeepSeek API Key**。
4.  保存文件。

### 第二步：加载插件
1.  打开 Chrome 浏览器，访问 `chrome://extensions/`。
2.  开启右上角的 **“开发者模式”**。
3.  点击左上角的 **“加载已解压的扩展程序”**。
4.  选择本项目文件夹 (`GOODHR_2.0`)。

## 💻 使用指南 (User Guide)

1.  **进入页面** 打开 BOSS直聘 **[推荐牛人]** 列表页，按 `F5` 刷新页面确保脚本注入。

2.  **启动控制台** 点击浏览器右上角的插件图标，打开控制面板。

3.  **设置岗位（可选）** 虽然 AI 内置了通用标准，但你可以添加岗位关键词来辅助“本地快筛”。
    * 输入岗位名称（如“亚马逊运营”）并点击添加。
    * (可选) 添加“包含关键词”或“排除关键词”来控制初步流量。

4.  **开始运行** 点击 **[▶ 开始 AI 筛选]** 按钮。
    * **观察**：你会看到页面自动滚动，侧边栏自动打开/关闭。
    * **标签**：每个候选人头像上会出现 AI 的评分标签（绿色=通过，灰色=淘汰）。
    * **日志**：按 `F12` 打开浏览器控制台 (`Console`) 可查看 AI 的详细思考过程。
